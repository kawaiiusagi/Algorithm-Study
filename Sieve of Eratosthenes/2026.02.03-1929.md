# 1929 소수 구하기
[백준 1929 소수 구하기](https://www.acmicpc.net/problem/1929)

## 문제 요약
두 정수 M과 N 사이에 존재하는 모든 소수를 구하여 출력하는 문제로, 에라토스테네스의 체를 이용해 효율적으로 소수를 판별하는 것이 핵심이다.

### 코드
```cpp
#include <iostream>
#include <vector>

using namespace std;

int main()
{
	int m, n;
	cin >> m >> n;

	vector<bool> t(n + 1, true);
	t[1] = false;

	for (int i = 2; i * i <= n; ++i) {
		if (t[i] == false) {
			continue;
		}

		for (int j = i * i; j <= n; j += i) {
			t[j] = false;
		}
	}

	for (int i = m; i <= n; ++i) {
		if (t[i]) {
			cout << i << "\n";
		}
	}

	return 0;
}
```
---
- `bool` 배열을 활용하여 소수 여부를 관리함으로써, 조건 분기 없이 빠르고 간결하게 알고리즘을 구현할 수 있었다.
- 이번 문제를 통해 **에라토스테네스의 체**라는 소수 판별 알고리즘을 처음 접하게 되었고, 단순 반복문보다 훨씬 효율적인 방식이라는 것을 이해하게 되었다.
