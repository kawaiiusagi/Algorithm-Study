# 1912번 연속 합
[백준 1912번 연속 합](https://www.acmicpc.net/problem/1912)

### 문제 요약
정수로 이루어진 수열이 주어질 때, 연속된 부분 수열 중 합이 가장 큰 값을 구하는 문제이다

### 코드
```cpp
#include <iostream>
#include <vector>

using namespace std;

int main()
{
	int n, plus;
	cin >> n;
	
	vector<int> v(n);

	for (int i = 0; i < n; i++) {
		int x;
		cin >> x;
		v[i] = x;
	}
	
	int current = v[0];
	int best = v[0];

	for (int i = 1; i < n; i++) {
		current = max(v[i], current + v[i]);
		best = max(best, current);
	}

	cout << best;

	return 0;
}
```
---
- 처음에는 연속된 수의 개수를 기준으로 경우를 나누어 설계하려 했지만, 시간 복잡도가 커지고 구현 또한 깔끔하지 않아 다른 방향으로 접근하였다.
- 이후 각 위치에서의 “부분 최대값”을 기준으로 점화식을 세워 해결하는 방식으로 전환하였다.
- 구현 과정에서 전체 최대값과 현재 위치까지의 부분 최대값을 혼동하여 잘못 갱신하는 실수가 있었고, 그로 인해 결과가 비정상적으로 나오는 오류를 겪었다.
- 이를 통해 DP 문제에서는 **현재 상태(부분 최대)**와 **전체 결과(전체 최대)**를 명확히 분리하여 관리하는 것이 중요하다는 점을 다시 한 번 느꼈다.
- 단순해 보이지만 상태 정의가 명확하지 않으면 쉽게 실수할 수 있는 문제라고 생각한다.

### 이 문제에서 중요한 점
- 연속 구간의 최대 합을 구하는 점화식 설계
- 부분 최대값과 전체 최대값을 분리하여 관리하는 구현 방식
