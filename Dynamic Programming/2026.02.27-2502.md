# 2502 떡 먹는 호랑이

### 문제 요약
날짜 D와 총 떡의 개수 K가 주어질 때, 1일째와 2일째에 준 떡의 개수를 구하는 문제이다.

### 코드
```cpp
#include <iostream>
#include <vector>
#include <algorithm>

using namespace std;

int main()
{
	int d, k, p = 0;
	cin >> d >> k;
	
	vector<string> v(d+1);

	v[1] = "a";
	v[2] = "b";

	for (int i = 3; i <= d; i++) {
		v[i] = (v[i - 2] + v[i - 1]);
	}

	int n1 = count(v[d].begin(), v[d].end(), 'a');
	int n2 = count(v[d].begin(), v[d].end(), 'b');

	for (int a = 1; a <= k; a++) {  
		for (int b = a + 1; b <= k; b++) {
			if ((n1 * a) + (n2 * b) == k && a < b) {
				cout << a << "\n" << b;
				return 0;
			}
		}
	}
}
```
- 처음에는 동적 프로그래밍 방식을 활용해 앞에서부터 값을 만들어가는 방식으로 접근하였다.
- 마지막 날의 값을 점화식 형태로 구성한 뒤, 계수를 찾아 1일째와 2일째 값을 구하려고 했다.
- 초반에는 계수를 찾기 위해 `find`를 활용하려 했으나, `find`는 단순히 위치를 반환하는 함수라는 점을 제대로 인식하지 못해 혼선이 있었다.
- 이후 `v[d-1].find("a");`와 같은 방식을 활용하여 필요한 정보를 찾을 수 있다는 점을 새롭게 이해하게 되었다.
- 다만 전체적으로 코드가 직관적이지 않고 다소 복잡하게 느껴져, 이후에는 점화식의 계수를 직접 정수 배열로 관리하는 방식 등 더 간결한 구조로 개선해보고 싶다는 생각이 들었다.

### 이 문제에서 중요한 점
- 점화식을 기반으로 한 동적 프로그래밍 사고
- 마지막 값을 기준으로 초기 값을 역추적하는 구현 방식
