# 9086 문자열
### 문제 요약
문자열을 입력으로 주면 문자열의 첫 글자와 마지막 글자를 출력하는 프로그램을 작성하는 것.

```cpp
#include <iostream>
#include <string>

using namespace std;

int main()
{
	int test;
	cin >> test;

	while (test--) {
		string s;
		cin >> s;

		cout << s[0] << s[s.length() - 1] << endl;
	}

	return 0;
}
```
- 첫 cpp 코드!! c에서 넘어와 적응하는데에 조금 애를 먹었다.
- 초반에 입력을 모두 저장하고 한 번에 출력을 꺼내는 것으로 문제를 잘못 이해하여 코드를 복잡하게 만들었었다.
- 그러나 문제를 제대로 이해한 후에는 for 문을 여러 번 쓰기보다는 어짜피 입력한 횟수만큼 전부 실행해야하는 것이기 때문에
while 문을 사용하여 코드를 더욱 간편하게 줄여주었다.

### 내가 생각하는 문제의 핵심
- 문자열과 반복문에 대한 이해도
