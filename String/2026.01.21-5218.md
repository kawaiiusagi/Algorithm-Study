# 5218 알파벳 거리
[백준 5218번 알파벳 거리](https://www.acmicpc.net/problem/5218)

### 문제 요약
각 케이스마다 각 글자의 알파벳 거리를 공백으로 구분해 출력한다.

### 코드
```cpp
#include <iostream>
#include <vector>

using namespace std;

int main()
{
	int test;
	cin >> test;

	while (test--) {
		string text1, text2;
		cin >> text1 >> text2;

		vector<int> v;

		for (int j = 0; j < text1.length(); j++) {
			int dist = text2[j] - text1[j];
			if (dist < 0) dist += 26;
			v.push_back(dist);
		}

		cout << "Distances: ";
		for (int x : v) {
			cout << x << " ";
		}
		cout << endl;

	}

	return 0;
}
```
- vector 사용이 아직 익숙하지 않아 초기 구현 단계에서 벡터에 원소를 추가하는 과정을 놓치는 실수가 있었다. 이 경험을 통해 C++에서 컨테이너를 사용할 때 데이터 삽입과 접근 흐름을 명확히 이해하는 것의 중요성을 인식하게 되었다.
- 이번 문제를 풀며 (int x : v) 형태의 범위 기반 for문을 새롭게 학습했는데, 이는 벡터에 저장된 원소를 하나씩 순회하며 사용할 수 있는 문법으로, 기존 인덱스 기반 반복문보다 코드의 가독성과 안정성을 높여준다는 점에서 인상 깊었다.
