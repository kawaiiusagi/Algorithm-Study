# 1159 농구 경기
[백준 1159번 농구 경기](https://www.acmicpc.net/problem/1159)

### 문제 요약
선수들의 성 첫 글자를 기준으로 같은 글자가 5번 이상 등장하면 해당 글자들을 사전순으로 출력하고, 없으면 PREDAJA를 출력하는 문제이다.

### 코드
```cpp
#include <iostream>
#include <string>
#include <vector>

using namespace std;

int main()
{
	int num;
	cin >> num;

	string name;

	vector<string> v;
	vector<char> first;
	vector<char> result;

	//v에서 각 첫번째 숫자만을 추출해서 개수를 파악하고 알파벳 순으로 정렬

	for (int i = 0; i < num; i++) {
		cin >> name;
		v.push_back(name);
	}

	for (string s : v) { //자료형에 주의하기
		first.push_back(s[0]);
	}
	//아스키 코드를 이용해서 개수를 검사하기? 음.
	for (char ch = 'a'; ch <= 'z'; ch++) {
		int cnt = 0;

		for (char c : first) {
			if (c == ch) cnt++;
		}

		if (cnt >= 5) {
			result.push_back(ch);
		} //result의 길이가 0이면 PREDAJA
	}
	if (result.empty()) {
		cout << "PREDAJA";
	}
	else {
		for (char c : result) {
			cout << c;
		}
	}
	//벡터는 바로 출력할 수 없어.
}
```
- 문제 해결을 위해 먼저 전체 흐름을 구상하였다. 각 과정에서 입력되는 **문자열의 길이가 서로 다르기 때문에**, 문자열 전체를 비교하는 대신 각 문자열의 첫 글자만을 추출하여 비교하는 방식이 적절하다고 판단하였다.
- 이에 따라 문자열들의 첫 글자를 별도의 벡터에 저장하고, 이후 해당 벡터를 대상으로 비교 및 개수 계산을 수행하도록 구조를 설계하였다.
- 구현 과정에서 처음에는 find 함수를 사용하려 했으나, **find가 벡터의 멤버 함수가 아니라는 점**을 인지하지 못해 오류가 발생하였다.
- 이를 해결하기 위해 알고리즘 라이브러리를 사용하지 않고, for 반복문을 통해 알파벳을 하나씩 순회하며 직접 개수를 세는 방식으로 코드를 수정하였다.
- 또한 **벡터는 cout으로 바로 출력할 수 없으므로**, 반복문을 통해 각 원소를 하나씩 출력해야 한다는 점을 다시 한 번 확인할 수 있었다.

### 이 문제에서 중요한 점
- 문제의 조건을 분석하여 적절한 알고리즘 구조를 설계하는 과정
- 문자열과 벡터의 특성을 이해하고 이를 상황에 맞게 활용하는 능력
