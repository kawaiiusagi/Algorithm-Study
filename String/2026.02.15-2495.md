# 백준 2495 연속구간

### 문제 요약
3개의 숫자 문자열이 주어질 때,  
각 문자열에서 같은 숫자가 연속으로 등장하는 최대 길이를 구하는 문제이다.

### 코드
```cpp
#include <iostream>
#include <string>

using namespace std;

int main() {
    for (int i = 0; i < 3; i++) {
        string s;
        cin >> s;

        int maxLen = 1;
        int curLen = 1;

        for (int j = 1; j < s.length(); j++) {
            if (s[j] == s[j - 1]) {
                curLen++;
            } else {
                curLen = 1;
            }

            if (curLen > maxLen) {
                maxLen = curLen;
            }
        }

        cout << maxLen << endl;
    }

    return 0;
}
```
---
### 풀이 과정 및 느낀 점

- 문자열을 순회하면서 이전 문자와 현재 문자를 비교하는 방식으로 해결하였다.
- 현재 연속 길이를 저장하는 curLen과, 최댓값을 저장하는 maxLen을 분리하여 관리하였다.
- 같은 문자가 이어질 경우 curLen을 증가시키고, 다른 문자가 나오면 1로 초기화하였다.
- 반복문 안에서 매번 maxLen을 갱신하여 가장 긴 연속 구간을 구했다.
- 별도의 자료구조 없이도 상태 변수 두 개만으로 깔끔하게 구현할 수 있는 문제였다.

### 중요한 포인트
문자열을 순차적으로 비교하는 구현 방식

현재 상태와 전체 최댓값을 분리하여 관리하는 사고
