# 9086 문자열
[백준 9086 문자열](https://www.acmicpc.net/problem/9086)
### 문제 요약
문자열을 입력으로 주면 문자열의 첫 글자와 마지막 글자를 출력하는 프로그램을 작성하는 것.

```cpp
#include <iostream>
#include <string>

using namespace std;

int main()
{
	int test;
	cin >> test;

	while (test--) {
		string s;
		cin >> s;

		cout << s[0] << s[s.length() - 1] << endl;
	}

	return 0;
}
```
- 첫 C++ 코드로, C에서 넘어오며 문법과 라이브러리에 적응하는 데 다소 애를 먹었다.
- 처음에는 입력을 모두 저장한 뒤 한 번에 출력하는 방식으로 문제를 잘못 이해해 코드가 불필요하게 복잡해졌다.
- 하지만 문제를 다시 읽고, 입력된 횟수만큼 동일한 작업을 반복하면 된다는 점을 파악한 뒤
  for문을 여러 번 사용하는 대신 while문을 활용해 코드를 더 간결하게 수정했다.

### 내가 생각하는 문제의 핵심
- 문자열과 반복문에 대한 이해도
