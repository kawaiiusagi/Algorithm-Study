# 11365 !밀비 급일
[백준 11365번 밀비 급일](https://www.acmicpc.net/problem/11365)
### 문제 요약
입력받은 문자열(암호)를 거꾸로 출력하도록 하는 것. 마지막 줄에는 "END"가 주어진다.(END는 해독X)

### 코드
```cpp
#include <iostream>
#include <string>

using namespace std;

int main()
{
	string code;

	while (true) {
		getline(cin, code, '\n');

		if (code == "END") {
			break;
		}

		for (int j = code.length() - 1; j >= 0; j--) {
			cout << code[j];
		}
		cout << '\n';
	}

	return 0;
}
```
- 이번에는 운이 좋게 코드가 정상적으로 작동했지만, 처음 작성했던 코드에서는 논리 구조에서 오류가 있었다.
- 초기 코드에서는 반복 조건을 `while (code != "END")`으로 설정했는데, 이 방식은 __입력을 받기 전에 조건을 먼저 검사하게 되어__ 논리 흐름상 어색한 구조가 된다.
- 이 문제의 올바른 흐름은 __입력 → 종료 조건 판단 → 처리__ 이므로, 먼저 getline으로 문자열을 입력받고, 그 후 "END"인지 확인해야 한다.
- 이를 해결하기 위해 반복 조건을 while (true)로 바꾸고, 입력 직후 "END" 여부를 검사하여 break로 반복문을 종료하도록 수정했다. 이렇게 구조를 변경함으로써 논리적으로 자연스럽고 안전한 코드가 되었다.

### 이 문제에서 중요한 포인트
- 문자열에 공백이 포함되므로 `getline`을 사용
- **입력 -> 판단 -> 처리** 순서를 지키는 것이 중요함.
