# 11365 !밀비 급일
[백준 11365번 밀비 급일](https://www.acmicpc.net/problem/11365)
### 문제 요약
입력받은 문자열(암호)를 거꾸로 출력하도록 하는 것. 마지막 줄에는 "END"가 주어진다.(END는 해독X)

### 코드
```cpp
#include <iostream>
#include <string>

using namespace std;

int main()
{
	string code;

	while (true) {
		getline(cin, code, '\n');

		if (code == "END") {
			break;
		}

		for (int j = code.length() - 1; j >= 0; j--) {
			cout << code[j];
		}
		cout << '\n';
	}

	return 0;
}
```
- 이번에는 운이 좋게 코드가 돌아갔었지만, 처음에 짰던 코드에서는 논리 구조에서 오류가 있었다.
- `getline`(입력)을 판단보다 먼저 받았어야했지만, 처음에 코드를 `while(code != "END")`로 작성하여
판단 -> 입력 순으로 되어 논리적으로 어색한 코드가 되었다.
- 따라서 입력받은 `code`값이 `"END"`가 아닐 경우에는 계속 반복문이 진행될 수 있도록 `true`를 사용해
코드를 고쳐주었다.
