# 10828 스택
[백준 10828번 스택](https://www.acmicpc.net/problem/10828)

### 문제 요약
정수를 저장하는 스택을 구현하여 입력으로 주어지는 명령(push, pop, size, empty, top)을 처리하고, 출력이 필요한 경우 그 결과를 한 줄씩 출력하는 문제이다

### 코드
```cpp
#include <iostream>
#include <stack>
#include <string>

using namespace std;

int main()
{
    stack<int> s;

    int n, num;
    string prom;
    cin >> n;

    for (int i = 0; i < n; i++) {
        cin >> prom;

        if (prom == "push") {
            cin >> num;
            s.push(num);
        }
        else if (prom == "top") {
            if (s.empty()) {
                cout << "-1" << endl;
            }
            else {
                cout << s.top() << endl;
            }
        }
        else if (prom == "size") {
            cout << s.size() << endl;
        }
        else if (prom == "pop") {
            if (s.empty()) {
                cout << "-1" << endl;
            }
            else {
                cout << s.top() << endl;
                s.pop();
            }
        }
        else if (prom == "empty") {
            cout << (s.empty() ? 1 : 0) << endl;
        }
    }
    return 0;
}
```
- 이 문제를 해결하는 과정에서 처음에는 cin >> prom >> num 형태로 명령어와 숫자를 동시에 입력받았으나, push 명령을 제외한 대부분의 명령은 추가 입력을 필요로 하지 않는다는 점을 고려하지 못해 입력 처리 과정에서 오류가 발생하였다. 이를 통해 명령어별 입력 형식을 정확히 구분하는 것의 중요성을 깨달았다.
- 또한 조건문을 반복적으로 나열하기보다는, else if 구조를 활용하여 코드를 보다 간결하고 직관적으로 구성할 필요성을 느꼈다.

### 이 문제에서 중요한 점
- 스택의 기본 연산(push, pop, top, size, empty)에 대한 이해
- 명령어에 따라 입력 처리 방식이 달라진다는 점을 고려한 구현 능력
- 조건문 구조를 정리하여 가독성과 안정성을 높이는 코드 작성
