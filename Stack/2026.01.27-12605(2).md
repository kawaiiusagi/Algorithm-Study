## sstream
-> 단어별로 저장하는 알고리즘을 더욱 간편하게 구현할 수 있는 방법

```cpp
#include <sstream>
#include <iostream>
#include <string>
#include <vector>

using namespace std;

int main() {
	string s;
	cin >> s;
	getline(cin, s);

	vector<string> split;
	stringstream ss(s);
	string word;

	while (ss >> word) {
		split.push_back(word);
	}

	while (getline(ss, word, ' ')) {
		split.push_back(word);
	}

	for (int i = 0; i < split.size(); i++) {
		cout << split[i] << "\n";
	}


}
```
### istringstream(입력용)
문자열 -> `cin`처럼 읽기
```cpp
#include <iostream>
#include <sstream>
using namespace std;

int main() {
    string s = "10 20 30";
    istringstream iss(s);

    int a, b, c;
    iss >> a >> b >> c;

    cout << a + b + c; // 60
}
```
(iss = 그냥 변수 이름, 괄호 안에 있는 s는 iss에 넣을 문자열을 의미한다.)

### ostringstream(출력용)
`cout`처럼 문자열 만들기
```cpp
#include <iostream>
#include <sstream>
using namespace std;

int main() {
    ostringstream oss;
    int x = 5, y = 7;

    oss << x << " + " << y << " = " << (x + y); //이 부분은 출력 내용을 쌓아두는 것

    string result = oss.str(); // 여기서 oss에 쌓아둔 내용들을 문자열로 꺼내달라는 의미이다.
    cout << result;
}
```

### stringstream(입력과 출력 둘 다)
```cpp
stringstream ss;
ss << "123"; // ss에 123이라는 문자열을 스트림 통 안에 집어넣는다.
int n;
ss >> n; // `ss`안에 들어 있는 문자열을 입력처럼 읽어서 `int n`에 넣는다.
```
