# 10789 세로읽기  
[백준 10789번 세로읽기](https://www.acmicpc.net/problem/10789)

### 문제 요약
총 5줄의 문자열이 주어지고, 각 줄의 같은 인덱스에 있는 문자들을 위에서 아래로 읽어 하나의 문자열로 출력하는 문제이다.  
문자열의 길이는 서로 다를 수 있으며, 해당 위치에 문자가 없는 경우는 건너뛴다.

### 코드
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s[5];

    for (int i = 0; i < 5; i++) {
        cin >> s[i];
    }

    for (int col = 0; col < 15; col++) {
        for (int row = 0; row < 5; row++) {
            if (col < s[row].length()) {
                cout << s[row][col];
            }
        }
    }
    return 0;
}
```
### 내가 느낀 점
- 처음에는 문자열을 단순히 이어 붙이는 문제라고 생각했지만, 실제로는 **세로 방향으로 접근해야 하는 문제**였다.
- 문자열마다 길이가 다르기 때문에 **범위를 벗어나는 접근을 막기 위한 조건문 처리의 중요성**을 느꼈다.
- 문자열을 2차원 배열처럼 생각하여 **행(row)과 열(col)의 개념으로 접근하는 연습**이 되었다.

### 이 문제에서 중요한 점
- 문자열 길이(`length()`) 확인
- 이중 반복문을 활용한 세로 방향 순회
- 없는 문자를 무시하는 조건 처리
