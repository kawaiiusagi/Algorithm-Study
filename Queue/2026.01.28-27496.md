# 27496 발머의 피크 이론
[백준 27496 발머의 피크 이론](https://www.acmicpc.net/problem/27496)

### 문제 요약
시간 순서대로 측정된 **알코올 농도 값**이 주어질 때, 연속된 구간의 합이 발머의 피크 이론에서 정의한 기준 범위에 해당하는 경우의 수를 구하는 문제이다.

### 코드
```cpp
#include <iostream>
#include <queue>
#include <string>
#include <vector>

using namespace std;

int main()
{
	int n, l;
	cin >> n >> l;
	cin.ignore();

	vector<int> temp;
	queue<int> now;
	vector<int> res;

	int result = 0;
	int time = 0;

	for (int i = 0; i < n; i++) {
		int num;
		cin >> num;

		if (now.size() >= l) {
			result -= now.front();
			now.pop();
		}
		now.push(num);
		result += num;

		if (result >= 129 && 138 >= result) {
			time += 1;
		}
	}

	cout << time;

	return 0;
}
```
- 이 문제를 풀면서 전체적인 알고리즘 구상은 비교적 수월했으나, 이를 코드로 옮기는 구현 과정에서 어려움을 겪었다.
- 조건에 해당하는 값만 `Queue`에 넣어 관리하고자 했지만, 큐의 동작 방식에 대한 이해가 부족해 의도한 대로 구현되지 않았다.
- 구현 도중 불필요한 코드들을 추가하면서 전체 구조가 복잡해졌고, 그 결과 처음에 짰던 코드는 최적화와는 다소 거리가 있는 코드가 되었다는 아쉬움이 남았다.
