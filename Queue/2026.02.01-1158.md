# 1158 요세푸스 문제
[백준 1128번 요새푸스 문제](https://www.acmicpc.net/problem/1158)

### 문제 요약
원형으로 배치된 사람들 중에서 K번째 사람을 반복적으로 제거하여 제거되는 순서를 < > 형식으로 출력하는 시뮬레이션 문제이다.

### 코드
```cpp
#include <iostream>
#include <queue>

using namespace std;

int main() {
    int n, k;
    cin >> n >> k;

    queue<int> q;

    for (int i = 1; i <= n; i++) {
        q.push(i);
    }

    cout << "<";

    while (q.size() > 1) {
        for (int i = 0; i < k - 1; i++) {
            q.push(q.front());
            q.pop();
        }
        cout << q.front() << ", ";
        q.pop();
    }

    cout << q.front() << ">";
    return 0;
}
```
### 풀이 과정 및 느낀 점
- 요세푸스 문제는 구조 자체는 단순하지만, 구현 과정에서 인덱스와 자료구조의 상태 변화를 정확히 관리하지 않으면 쉽게 꼬일 수 있는 문제라고 느꼈다.
- 처음에는 단순히 규칙대로 사람을 제거하면 된다고 생각했지만, 원형 구조를 직접 구현하려다 보니 인덱스 계산이 복잡해지고 실수가 잦았다.
- 이 문제에서는 **현재 위치에서 K번째를 찾은 뒤 제거하고, 그 다음 위치를 기준으로 다시 시작한다는 흐름**을 명확히 유지하는 것이 중요했다.
- 큐를 사용해 앞의 원소를 뒤로 보내는 방식으로 원형 구조를 간접적으로 구현하니, 인덱스 계산 없이도 문제를 해결할 수 있었다.
- 결과를 출력할 때는 문제에서 요구하는 출력 형식(`<`, `,`, `>`)을 정확히 맞추는 것 또한 신경 써야 했다.

---

### 이 문제에서 중요한 점
- 원형 구조를 직접 구현하기보다는, **큐를 이용해 순환을 시뮬레이션하는 사고**
- 제거 이후의 기준 위치를 정확히 유지하는 로직 설계
- 출력 형식에 대한 세심한 처리
