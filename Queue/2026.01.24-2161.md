# 2161 카드1
[백준 2161번 카드1](https://www.acmicpc.net/problem/2161)

### 문제 요약
- 카드가 1부터 N까지 순서대로 놓여 있을 때, 맨 위 카드를 버리고 그 다음 카드를 맨 아래로 옮기는 과정을
복하여 버려지는 카드의 순서를 출력하는 문제이다.

### 코드
```cpp
#include <iostream>
#include <queue>

using namespace std;

int main()
{
	int n;
	cin >> n;

	queue<int> q;

	for (int i = 1; i <= n; i++) {
		q.push(i);
	}
	while (q.size() > 1) {
		cout << q.front() << " ";
		q.pop();
		int x = q.front();
		q.pop();
		q.push(x);
	}
	cout << q.front();

	return 0;
}
```
- 큐를 활용하는 과정에서 반복 조건을 `q`가 아닌 `n`을 기준으로 설정하는 실수를 하여, 의도한 동작과
다른 결과가 발생하였다. 이를 통해 **반복문의 조건은 자료구조의 상태를 기준으로 설정해야 한다는 점**을
다시 한 번 인식하게 되었다.
- 또한 큐의 크기가 1이 될 때까지 연산을 반복한 뒤, 마지막에 남은 값을 출력해야 한다는 조건을 코드로 구현하는
과정에서 알고리즘 흐름을 구상하는 데 많은 시간이 소요되었다.

### 이 문제에서 중요한 점
- 큐의 기본 동작(삽입, 삭제, 크기 변화)에 대한 이해
- 반복문의 종료 조건을 큐의 상태(`size`)와 정확히 연결하여 사고하는 능력
