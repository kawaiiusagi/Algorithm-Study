# 11399 ATM
[백준 11399번 ATM](https://www.acmicpc.net/problem/11399)
### 문제 요약
각 사람이 돈을 인출하는데 필요한 시간의 합의 최솟값을 출력하도록 하는 것.
### 코드
```cpp
#include <iostream>
#include <string>
#include <vector>

using namespace std;

int main()
{
	int n, sum = 0;
	int total = 0;

	vector<int> v;

	cin >> n;

	for (int i = 0; i < n; i++) {
		int x;
		cin >> x;
		v.push_back(x);
	}

	for (int j = 1; j < n; j++) {
		int key = v[j];
		int k = j - 1;

		while (k >= 0 && v[k] > key) {
			v[k + 1] = v[k];
			k--;
		}
		
		v[k + 1] = key;

	}

	for (int m = 0; m < n; m++) {
		sum += v[m];
		total += sum;
	}

	cout << total << endl;

	return 0;
}
```
- C++에서 `vector`를 활용한 배열 사용이 처음이라 기본적인 문법이었음에도 적응하는 데 시간이 걸렸다.
- 또한 아직 정렬 알고리즘이 완전히 체화하지 못한 상태라, 코드를 짜는데에 조금 애를 먹었다.
- 특히 누적 계산을 구현했다고 생각했음에도 불구하고, 실제로는 누적합이 아닌 단순 합만 계산되어 예상과 다른 값이 출력되었는데, 이 부분이 문제 해결 과정에서 가장 어려웠던 지점이었다.
### 내가 생각하는 이 문제에서 중요한 포인트
- 이 문제도 역시나 **정렬 알고리즘의 중요성**을 잘 보여주는 문제이다.
- 각 사람이 돈을 인출하는 데 필요한 시간의 합이 최소가 되기 위해서는, 인출 시간이 짧은 사람부터 **오름차순**으로 정렬해야 한다.
- 그 이유는 **누적 계산** 때문이다. 앞에 있는 사람의 인출 시간은 뒤에 오는 모든 사람의 대기 시간에 반복해서 더해지는데, 이 값이 작을수록 전체 누적합이 작아진다.
따라서 인출 시간이 짧은 사람을 앞에 배치할수록 전체 시간이 최소가 된다.
