# 1021 시퀀스 컨테이너
[백준 1021 시퀀스 컨테이너](https://www.acmicpc.net/group/workbook/view/16151/50990)

### 문제 요약
양방향 순환 큐에서 원하는 숫자들을 순서대로 꺼내기 위해 왼쪽·오른쪽 회전을 최소로 수행하여 총 회전 횟수의 최솟값을 구하는 문제이다.

### 코드
```cpp
#include <iostream>
#include <queue>
#include <deque>

using namespace std;

int main()
{
    int n, m;
    cin >> n >> m;

    int cnt = 0;

    queue<int> q;
    deque<int> deq;

    for (int i = 0; i < m; i++) {
        int x;
        cin >> x;
        q.push(x);
    }

    for (int j = 1; j <= n; j++) {
        deq.push_back(j);
    }

    while (!q.empty()) {
        int target = q.front();
        q.pop();

        int pos = 0;
        while (deq[pos] != target) {
            pos++;
        }

        if (pos <= deq.size() / 2) {
            while (deq.front() != target) {
                deq.push_back(deq.front());
                deq.pop_front();
                cnt++;
            }
        }
        else {
            while (deq.front() != target) {
                deq.push_front(deq.back());
                deq.pop_back();
                cnt++;
            }
        }

        deq.pop_front();
    }

    cout << cnt;
    return 0;
}
```
- 이번 문제를 통해 `deque` 자료구조를 처음 사용해 보았다. `deque`는 앞과 뒤 양쪽에서 원소의 삽입·삭제가 가능하여, 회전 연산이 필요한 문제에서 효율적으로 사용할 수 있었다.
- 구현 과정에서 처음에는 `deq.back()` 값을 임시 변수에 저장한 뒤 `push_front`하는 방식으로 작성했으나, 이를 `deq.push_front(deq.back());` 한 줄로 정리하여 코드를 간결하게 만들었다.
- 이 문제를 통해 `list`, `vector`, `deque`의 차이를 다시 정리하게 되었고, 앞으로는 자료구조의 특성에 맞게 선택하여 불필요한 연산을 줄이는 방향으로 구현해야겠다고 느꼈다.

### Key Points
- `deque`를 이용한 양방향 구
