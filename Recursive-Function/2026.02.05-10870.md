# 피보나치수 5
[백준 10870번 피보나치 수 5](https://www.acmicpc.net/problem/10870)

### 문제 요약
피보나치 수열이 주어질 때, N번째 피보나치 수를 재귀 또는 반복문을 이용해 계산하는 문제이다.

### 코드
```cpp
#include <iostream>

using namespace std;

int sum(int p)
{
	if (p <= 2) {
		if (p == 0) return 0;
		return 1;
	}
	return sum(p - 1) + sum(p - 2);
	
}

int main()
{
	int n;
	cin >> n;

	int x = sum(n);
	cout << x;

	return 0;
}
```
---
- 재귀 함수를 이용한 사고가 아직 익숙하지 않아 문제를 이해하고 구현하는 과정에서 다소 어려움을 느꼈다.
- `return sum(p - 1) + sum(p - 2)`라는 형태는 자연스럽게 떠올렸지만, 함수가 호출되고 종료되는 전체 흐름을 명확히 이해하지 못해 작동 원리를 파악하는 데 시간이 걸렸다.
- 이번 문제를 통해 재귀 함수는 단순히 식을 외우는 것이 아니라, **기저 조건과 함수 호출의 흐름을 함께 이해해야 한다는 점**이 중요하다는 것을 느꼈다.

### 이 문제에서 중요한 점
- 재귀 함수의 호출 구조와 종료 조건을 중심으로 사고하는 능력
