# 1427번 소트인사이드

### 문제 요약
어떤 수가 주어진다면 그 수의 각 자리수를 내림차순으로 정리해보도록 하는 것

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main() {
    int n;
    int arr[10];
    int cnt = 0;

    scanf("%d", &n);

    while (n > 0) {
        arr[cnt++] = n % 10;
        n /= 10;
    }

    for (int i = 1; i < cnt; i++) {
        int key = arr[i];
        int j = i - 1;

        while (j >= 0 && arr[j] < key) {
            arr[j + 1] = arr[j];
            j--;
        }
        arr[j + 1] = key;
    }

    for (int i = 0; i < cnt; i++) {
        printf("%d", arr[i]);
    }

    return 0;
}

```
- 문제를 보며 먼저 숫자를 각 자리수로 분리해 배열에 저장해야겠다고 판단했다.
- 처음에는 복합 대입 연산자(`/=`)를 떠올리지 못해 코드가 다소 복잡해졌지만,
  이후 while 반복문과 복합 대입 연산자를 활용해
  자릿수 분리 과정을 훨씬 간결하게 수정할 수 있었다.
  
### 내가 생각하는 이 문제의 핵심
- `n % 10`과 `n /= 10`을 이용한 자릿수 분리
- 정렬 알고리즘을 활용한 내림차순 정렬 구현
