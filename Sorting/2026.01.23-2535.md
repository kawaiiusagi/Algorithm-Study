# 2535 아시아 정보올림피아드
[백준 2535 아시아 정보올림피아드](https://www.acmicpc.net/problem/2535)

### 문제 요약
여러 나라 학생들의 성적이 주어질 때 성적 순으로 상위 3명을 선발하되, 한 나라에서 최대 2명까지만 메달을 받을 수 있도록 하여 금·은·동메달 수상자의 국가 번호와 학생 번호를 출력하는 문제이다.

### 코드
```cpp
#include <iostream>
#include <vector>
using namespace std;

class student {
private:
    int country, number, score;
public:
    void input() {
        cin >> country >> number >> score;
    }

    int getScore() const {
        return score;
    }

    int getCountry() const {
        return country;
    }

    int getNumber() const {
        return number;
    }
};

int main() {
    int num;
    cin >> num;

    vector<student> stud(num);

    for (int i = 0; i < num; i++) {
        stud[i].input();
    }

    for (int i = 1; i < num; i++) {
        student key = stud[i];
        int j = i - 1;

        while (j >= 0 && stud[j].getScore() < key.getScore()) {
            stud[j + 1] = stud[j];
            j--;
        }
        stud[j + 1] = key;
    }

    int countryCount[101] = {0};
    int cnt = 0;

    for (int i = 0; i < num && cnt < 3; i++) {
        int c = stud[i].getCountry();

        if (countryCount[c] < 2) {
            cout << stud[i].getCountry() << " " << stud[i].getNumber() << '\n';
            countryCount[c]++;
            cnt++;
        }
    }

    return 0;
}
```
- 이 문제를 해결하는 과정에서 여러 개의 정보를 반복적으로 입력받아 처리해야 했기 대문에, **관련 데이터를 하나의
데이터로 관리하기 위해** 클래스를 활용하기로 결정하였다.
- 클래스를 사용하는 과정에서 클래스 내부의 값을 외부에서 사용하기 위해서는 **반환 함수(getter)를 통해 값을
가져와야 한다는 점**을 처음에는 인지하지 못해, 클래스 내부 데이터에 접근하는 데 어려움을 겪었다.
- 또한 클래스(또는 구조체)를 사용해 객체를 선언할 때, **해당 클래스명을 자료형처럼 명시**해야 한다는 점을 놓쳐 초기 구현
과정에서 혼란이 있었다.
- 더불어 각 국가별로 메달 수상자를 최대 2명까지만 선발해야 한다는 조건을 코드로 논리화하는 과정에서 사고가
쉽지 않았으나, 조건 분기와 카운팅을 통해 해결할 수 있었다.

### 이 문제에서 중요한 점
- 여러 정보를 하나의 객체로 관리하는 **클래스/구조체의 활용 능력**
- 문제의 조건을 정확히 해석하고 알고리즘으로 구현하는 사고
