# 백준 2511 카드놀이

### 문제 요약
두 사람이 각각 10장의 카드를 내며, 각 라운드에서 더 큰 숫자를 낸 사람이 점수를 얻는다.  
모든 라운드가 끝난 뒤 총점을 비교하여 승자를 결정하는 문제이다.

### 코드

```cpp
#include <iostream>
#include <vector>

using namespace std;

int main()
{
    vector<int> num;

    int a = 0;
    int b = 0;

    bool lasta = true;
    bool same = false;

    for (int i = 0; i < 10; i++) {
        int n;
        cin >> n;
        num.push_back(n);
    }

    for (int j = 0; j < 10; j++) {
        int n;
        cin >> n;

        if (j == 9) {

            if (n > num[j]) {
                b += 3;
                lasta = false;
            }
            else if (n < num[j]) {
                a += 3;
            }
            else {
                a += 1;
                b += 1;
                same = true;
            }
            continue;
        }

        if (n > num[j]) {
            b += 3;
        }
        else if (n < num[j]) {
            a += 3;
        }
        else {
            a += 1;
            b += 1;
        }
    }
    
    cout << a << " " << b << endl;
    if (a > b) {
        cout << "A";
    }
    else if (b > a) {
        cout << "B";
    }
    else {
        if (lasta) {
            cout << "A";
        }
        else if (!lasta) {
            cout << "B";
        }
        else if (same) {
            cout << "D";
        }
    }

    return 0;
}
```
---

### 풀이 과정 및 느낀 점

- 각 라운드마다 점수를 즉시 누적하는 방식으로 구현하였다.
- 처음에는 무승부 조건과 마지막 승자 판정을 잘못 이해하여 수정하는 과정이 필요했다.
- 동점일 경우 마지막 라운드의 승자를 기준으로 결과를 출력해야 하므로, 이를 위해 `lasta` 변수를 사용하였다.
- 조건 분기가 많아질 수 있는 문제였지만, 점수 계산과 최종 판정을 분리하여 구현하려고 하였다.
- 단순 구현 문제일수록 조건을 정확히 해석하는 것이 중요하다고 느꼈다.

## 중요한 포인트

- 라운드별 점수 누적 처리
- 동점 시 마지막 결과를 고려한 조건 구현
