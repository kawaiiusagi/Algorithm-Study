# 2511 카드놀이
[백준 2511 카드놀이](https://www.acmicpc.net/problem/2511)

### 문제 요약
두 사람이 10장의 카드를 각각 내며 각 라운드에서 더 큰 숫자를 낸 사람이 점수를 얻고, 모든 라운드가 끝난 뒤 총점을 비교해 승자를 결정하는 문제이다.

### 코드
```cpp
#include <iostream>
#include <vector>

using namespace std;

int main()
{
    vector<int> num;

    int a = 0;
    int b = 0;

    char lastWinner = 'D';

    for (int i = 0; i < 10; i++) {
        int n;
        cin >> n;
        num.push_back(n);
    }

    for (int j = 0; j < 10; j++) {
        int n;
        cin >> n;

        if (n > num[j]) {
            b += 3;
            lastWinner = 'B';
        }
        else if (n < num[j]) {
            a += 3;
            lastWinner = 'A';
        }
        else {
            a += 1;
            b += 1;
        }
    }

    cout << a << " " << b << endl;

    if (a > b) {
        cout << "A";
    }
    else if (b > a) {
        cout << "B";
    }
    else {
        cout << lastWinner;
    }

    return 0;
}
```
---
### 풀이 과정 및 느낀 점
- 문제 자체는 크게 어렵지 않았지만, 처음에 무승부 조건을 정확히 이해하지 못해 수정하는 데 시간이 조금 걸렸다.
- 처음에는 **`bool` 변수를 활용하여 승패를 판단**하려고 했으나, 조건이 늘어나면서 오히려 코드가 복잡해졌다.
- 이후 각 라운드의 결과를 처리하면서 마지막에 이긴 사람을 저장하는 변수를 따로 두는 방식으로 수정하였고, 이 방법이 훨씬 간결하고 직관적이라는 것을 느꼈다.
- 그 결과 불필요한 **조건 분기와 코드 길이를 줄일 수 있었고, 전체 구조도 더 깔끔해졌다.**
- 단순 구현 문제일수록 **조건을 정확히 해석하고, 최소한의 변수로 구조를 설계**하는 것이 중요하다고 느꼈다.

### 이 문제에서 중요한 점
- 조건을 정확히 해석하고 구현하는 알고리즘 설계 능력
