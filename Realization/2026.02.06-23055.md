# 백준 23055 공사장 표지판

### 문제 요약
정수 N이 주어질 때, NxN 크기의 표를 출력하는 문제로,  
가장자리와 두 대각선 위치에 `*`을 출력하고 그 외에는 공백을 출력하는 구현 문제이다.


### 코드
```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cin >> n;

    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            if (i == 0 || i == n - 1 || j == 0 || j == n - 1
                || i == j || i + j == n - 1) {
                cout << "*";
            }
            else {
                cout << " ";
            }
        }
        cout << "\n";
    }
    return 0;
}
```
---

### 풀이 과정 및 느낀 점
- 이 문제는 출력 형식만 보면 복잡해 보였지만, 실제로는 좌표 `(i, j)`의 조건을 잘 나누어 생각하면 비교적 단순한 구현 문제였다.
- 처음에는 패턴을 직접 그려보며 규칙을 찾으려고 했는데, 이 과정에서 **가장자리 조건과 대각선 조건을 분리해서 생각하는 것**이 중요하다는 것을 느꼈다.
- 구체적으로는 다음 조건 중 하나라도 만족하면 `*`을 출력하도록 정리할 수 있었다.
  - 첫 번째 행 또는 마지막 행
  - 첫 번째 열 또는 마지막 열
  - `i == j` (왼쪽 위 → 오른쪽 아래 대각선)
  - `i + j == N - 1` (오른쪽 위 → 왼쪽 아래 대각선)
- 이처럼 조건을 명확히 나눈 뒤에는, 이중 반복문 안에서 `if`문 하나로 처리할 수 있어 코드가 깔끔해졌다.
- 출력 문제에서는 사소한 조건 하나를 놓치면 전체 출력이 틀어질 수 있기 때문에, **패턴 문제일수록 조건을 글로 먼저 정리하는 습관이 중요하다**고 느꼈다.

---

### 이 문제에서 중요한 점
- 2차원 좌표 `(i, j)`를 기준으로 조건을 나누는 사고
- 가장자리 조건과 대각선 조건을 분리하여 구현하는 방식
- 출력 문제에서 조건을 먼저 정리한 뒤 코드로 옮기는 습관
